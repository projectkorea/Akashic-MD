## CSS 적용 방법

**1. Add `<link />` in head tag**

```html
<head>
    <link rel="stylesheet" href="style.css" />
</head>
```

**2. Inline Style css**

-   html에서 tag에 `style=''` attribute을 이용해서 사용하는 경우가 있다.

```html
<p style="font-size: 32px;"></p>
```

-   마크업 문서에 스타일을 입히면 디버깅할 때 매우 불리하기 때문에, 특별한 경우 아니면 쓰지 말자.
