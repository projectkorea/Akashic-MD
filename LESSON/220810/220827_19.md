### CSS 문법

```css
선택자 {
  속성: 값;
}

selector {
  property: value;
}
```

줄바꿈 CSS 속성: `display`

```css
.laptop {
  display: block; 한 라인 독차지
}
```

```css
.laptop {
  display: inline;.
}
```

- 줄바꿈 안되고, 가로배치에만 쓰인다

- 자식까지 inline으로 만들어야, 전체가 가로정렬이 된다.

### `display: flex`

- flex 값은 부모한테 적용해야함.
